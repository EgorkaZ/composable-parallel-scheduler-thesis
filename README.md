# Fast work distribution for composable task scheduling engines

## Dependencies
- Cmake (>=3.15)
- Clang
- Gtest, google benchmark (will be built from source if not installed)

## Build & Run
```bash
make bench # runs benchmarks and saves results to ./bench_results
```
